How to Code in VirtueBar?

This might be a big question for you that "how to code in VirtueBar? " or "where to start learning? " .
You have come to the right place. The Official Wiki of VirtueBar handwritten by the developer and designer of the language.
Now without wasting our time, lets get started.

Insight on VirtueBar

VirtueBar is a simple, East-to-learn, high level programming language developed by a single person in python and bare code from scratch. It is in development and many features are yet to be added.

Installation

Before we jump right start into the coding part, lets see how we can install virtuebar interpreter and shell in your system.

Go to our Github Page or Download Page and download you version you want. After clicking on the downlaod button accordingly, a zip file will start downloading.



Go to where you have downloaded the file and right click on the zip file to extract the files inside.



Click on the Folder name VirtueBar and go inside it.



Now the important part: inside the folder, press the following keys on your keyboard.

Alt + F + S + A This will cause a popup window to appear like this:



Click on yes. Now it will open a powershell window like this:



Now type the following commands one after another:

cmd

and hit enter. Now you are in command prompt mode through powershel with administrive privilages. Before going forward, make sure you are in the VirtueBar folder containing the files shown in Picture 2. If you have then type the next command to successfully install the interpreter.

setup

Follow the commands and intructions in the setup command-line. If you see the text "Completed installation!" then you can rest assure, your program is successfully installed and now we can inject a viru- i mean you can use our language.
Have a happy coding session!

Grammer

Expression
=> KEYWORD:VAR IDENTIFIER Equals Expression
=> Comparison-Expression ((KEYWORD:[WITH|OR] Comparison-Expression) *Optional


Comparison Expression
=> Arithmetic-Expression ((DE|LT|GT|LTE|GTE|NT) Arithmetic-Expression) *Optional
=> INVER Comparison-Expression (NOT operations)


Arithmatic Expression
=> term ((PLUS | MINUS) term) *Optional


Term
=> factor ((MUL | DIV) factor) *Optional


Factor
=> (PLUS | MINUS) factor
Power


Power
=> atom (POWER factor) *Optional


Atom
=> INT|FLOAT|IDENTIFIER
LPAREN Expression RPAREN


If Expression
=> KEYWORD:IF Expression KEYWORD:ACT Expression
=> (KEYWORD:ELIF Expression KEYWORD:ACT Expression) *Optional and Unlimited
=> (KEYWORD:OTHER Expression) *Optional


For Loop Expression
=> KEYWORD:FOR IDENTIFIER Equals Expression KEYWORD:TO Expression KEYWORD:ACT Expression
=>KEYWORD:FOR IDENTIFIER Equals Expression KEYWORD:TO Expression (KEYWORD:STEP Expression) KEYWORD:ACT Expression


While Loop Expression
=> KEYWORD:WHILE Expression KEYWORD:ACT Expression
